## Задача 1
### Постановка задачи

> Ваша задача – посмотреть, какие значения принимает последовательность при разных a0 и n, вывести закономерность, по которой строится последовательность и запрограммировать ее самостоятельно.
> 
|      a0     |          n         |                Вывод                 |
|-------------|--------------------|--------------------------------------|
|      1      |         10         | 2, 4, 5, 10, 11, 22, 23, 46, 47, 94  |
|      3      |         10         | 4, 8, 9, 18, 19, 38, 39, 78, 79, 158 |

### Алогритм

Пусть ai - i-й член последовательности. 

a1 = a0+1

a2 = a1*2

a3 = a2+1

a4 = a3*2

Тогда если i - четное, то ai = a(i-1)*2, если i - нечетное, то ai = a(i-1)+1

### Программа
```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);
        int a0 = in.nextInt();
        int n = in.nextInt();

        for (int i = 0, a = a0; i < n; i++) {
            if(i%2==0) {
                a++;
                System.out.print(a + " ");
            }
            else{
                a*=2;
                System.out.print(a + " ");
            }
        }
    }
}
```

## Задача 2

### Постановка задачи

> С клавиатуры вводится заглавная буква латинского алфавита и число N (N < 10). Нужно вывести лесенку из букв, которая содержит до N ступеней: в первой строке одна буква, начиная с введенной, во второй – две следующие, в третьей – три следующие и т.д. Если алфавит заканчивается, построение лесенки останавливается на предыдущей полной ступени. Ступени с четным номером печатаются в обратном порядке.
>

### Алгоритм
last - количестово букв в уже построенных ступеньках.

Проверка возможности построения ступеньки: если текущая буква w + last не превышает 'Z', можно строить следующую ступень.

Постороение ступеньки: если ступенька нечетная, то выводим буквы в прямом порядке от w до (i-1); если ступенька четная, то выводим буквы в обратнои порядке от (w+i-1) до w. 

Обновляем начальную букву: w = w+i; переходим на новую строку для новой ступеньки.

### Программа
```java
import java.io.IOException;
import java.util.Scanner;

public class Main {
    public static void main(String[] args) throws IOException {
        Scanner in = new Scanner(System.in);
        char w = (char) System.in.read();
        int N = in.nextInt();
        int last = 0;
        for (int i = 1; i < N; i++) {
            if (w + last <= 'Z') {
                last = 0;
                if (i % 2 != 0) {
                    for (int j = 0; j < i; j++) {
                        if (w + j <= 'Z') {
                            System.out.print((char) (w + j));
                            last++;
                        }
                    }
                } else {
                    for (int j = i - 1; j >= 0; j--) {
                        if (w + j <= 'Z') {
                            last++;
                            System.out.print((char) (w + j));
                        }
                    }

                }
                w = (char) (w + i);
                System.out.println();
            }
        }
    }
}
```

## Задача 3

### Постановка задачи
>С клавиатуры вводятся целые числа. Вывести на экран количество тех первых введенных чисел, которые составляют арифметическую прогрессию. При обнаружении конца арифметической прогрессии остальные числа можно не вводить.
>

### Алгоритм
Начальное значение k = 2(первые два элемента последовательности). Находим разность двух первых введенных чисел - это и есть разность последовательности d. Далее находим разность между каждыми двумя элементами последовательности d1 и сравниваем ее с разностью d. Пока они равны увеличиваем счетчик k и продолжаем ввод чисел.

### Программа 
```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);
        int a; int k = 2;
        int a1 = in.nextInt();
        int b = in.nextInt();
        int d = b-a1; int d1 = d;
        while(d1==d){
            k++;
            a = in.nextInt();
            d1 = a-b;
            b = a;
        }
        k--;
        System.out.println(k);
    }
}
```

## Задача 4

### Постановка задачи
>Дан массив из n положительных чисел. Заменить каждое число в массиве на количество его делителей (включая 1 и само число) и вывести полученный массив.
>
Формат ввода:

Первая строка: число n

Вторая строка: n положительных целых чисел

### Алгоритм
Проходим по массиву, ищем количество делителей каждого элемента k и заменяем каждый элемент массива значением k.

### Программа
```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);
        int n = in.nextInt();
        int[] a = new int[n]; int k; int num;
        for (int i = 0; i < n; i++) {
            a[i] = in.nextInt();
        }
        for (int i = 0; i < n; i++) {
            k = 0; num = a[i];
            for (int j = 1; j < Math.sqrt(num); j++) {
                if(num%j==0)
                    k+=2;
            }
            if (Math.sqrt(num) == (int)Math.sqrt(num))
                k++;
            a[i] = k;
        }

        for (int i = 0; i < n; i++) {
            System.out.print(a[i] + " ");
        }
    }
}
```
